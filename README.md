# Packet-plugin

## 介绍

- **为对接NapCat.OneBot的BOT提供发送ProtoBuf包的能力**
- **提供了易用的等接口**
- **内置了发送按钮的能力(QQ 9.1.55以上可见)**

***NOTE: 请勿滥用该插件，使用此插件产生的一切后果由使用者承担***

## 安装
```
cd plugins
git clone https://gitee.com/HDTianRu/Packet-plugin
cd ..
pnpm install
```
